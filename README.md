# robo-advisor-etf

It is recommended to use a virtual environment
```
cd robo-advisor-etf/
pip3 install -U virtualenv
python3 -m virtualenv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

## Executing the Dashboard
```
cd robo_advisor_etf/
python dashboard.py 
```
